음성배치 (Voice Placement)
그림의 위치를 클릭하거나, 오른쪽 카드를 선택하면 각 위치의 음성 특성을 확인할 수 있습니다.

위치 선택
N — 비강 (Nasal cavity)
코 위쪽 공간. 비음 공명이 이루어지는 부위.
관련 음소: ㅁ · ㄴ · ㅇ (비음)
X — 정상 음성 배치
구강 중앙부. 건강한 발성의 기준 위치.
편안하고 자연스러운 공명 위치
A — 전 (Anterior)
입 앞쪽. 소리를 앞으로 띄우는 배치. 비음-유음 자극 훈련의 목표 위치.
밝고 앞쪽 공명 · 허밍 훈련 시 목표
B — 후 (Posterior)
인두/후두 쪽. 소리가 목 안으로 들어간 배치. 음성 과부하의 원인이 될 수 있음.
어둡고 후방 공명 · 개선이 필요한 위치
위치를 클릭하면 상세 설명이 표시됩니다.
🎵 음성 파일 추가 방법
각 위치 카드에 음성 예시를 연결하려면 selectZone() 함수 내부에 아래 코드를 추가하세요:
const audio = new Audio('/your-audio/zone_N.mp3'); audio.play();
파일명을 zone_N.mp3 / zone_X.mp3 / zone_A.mp3 / zone_B.mp3으로 지정하면 자동 연결됩니다.
